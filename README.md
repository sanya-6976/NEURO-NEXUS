🧠 NeuroNexus
NeuroNexus is an AI-powered productivity and assistance platform designed to bring multiple intelligent tools into a single, seamless user experience. The platform combines conversational AI, voice interaction, computer vision capabilities, and wellness-focused features to help users work smarter, learn faster, and stay productive.

🚀 Features

🤖 AI Hub
Centralized interface for interacting with AI-powered tools.
Supports intelligent conversations and task assistance.
Designed to enhance productivity and information access.

🎤 Voice Assistant
Enables voice-based interaction with the platform.
Provides hands-free access to AI functionalities.
Improves accessibility and user engagement.

👁️ Computer Vision
Integrates computer vision capabilities for image-based analysis and interaction.
Demonstrates the application of AI beyond text-based systems.
Supports future expansion into advanced visual intelligence features.

💙 Therapy Zone
Wellness-focused section designed to provide a supportive and calming user experience.
Encourages healthy digital interactions and user well-being.

📊 Interactive Dashboard
Modern and intuitive dashboard for navigating platform features.
Responsive design for smooth usage across devices.
Centralized access to all AI modules.
